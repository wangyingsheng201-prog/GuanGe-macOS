《观格 for macOS》版本更新日志

============================================================
1.0.0（2026-07-16）
============================================================

【全新 macOS 原生版本】
- 使用 Swift、SwiftUI 与 AppKit 重新实现，不依赖 Windows 组件或第三方运行库。
- 最低支持 macOS 13 Ventura。
- 生成 Universal 2 程序，同时支持 Apple Silicon 与 Intel Mac。
- 提供菜单栏常驻入口；关闭控制面板后参考线仍可继续工作。

【参考层】
- 新增透明、全屏、鼠标穿透的构图参考层，不影响播放器、浏览器和其他软件操作。
- 支持普通桌面空间与全屏应用空间。
- 支持全部显示器、主显示器和指定显示器。
- 所有设置即时生效，取消“应用”按钮。
- 参考线均为实线，不使用描边色。

【画幅比例】
- 支持适配屏幕、1:1、4:3、3:2、16:10、16:9、18:9、21:9。
- 支持电影画幅 1.85:1、2.35:1、2.39:1。
- 支持竖屏画幅 4:5、3:4、9:16。

【构图参考线】
- 三分法/九宫格、二等分、四等分网格、五等分网格。
- 黄金分割网格、中心十字。
- 左右黄金三角形。
- 四个方向的黄金螺旋。
- 动态对称。
- 对角线可独立开启或关闭。

【安全框】
- 安全框可独立开启或关闭，范围可按 1%–99% 调整，默认 90%。
- 安全框颜色可单独设置。
- 绘制逻辑改为四边相同的像素内距，避免宽画幅或竖画幅中横向与纵向边距不同。

【线条外观】
- 支持自定义主线颜色、安全框颜色、线宽与透明度。
- 提供八种轮巡预设颜色。

【截图】
- 支持全局快捷键截屏，截图包含当前屏幕画面和参考线。
- 截屏时自动隐藏控制面板和打赏窗口，完成后恢复。
- 支持自定义截图目录、浏览目录和在访达中打开目录。
- 多显示器模式可为每台显示器分别保存截图。

【快捷键】
- 支持显示/隐藏参考线、截屏、显示/隐藏控制面板。
- 支持轮巡切换画幅比例：默认 Shift + ↑ / Shift + ↓。
- 支持轮巡切换主构图线：默认 Control + ↑ / Control + ↓。
- 支持轮巡切换线条颜色：默认 Option + ↑ / Option + ↓。
- 所有快捷键均可由用户直接单击快捷键框并按下新组合完成替换。

【控制面板与其他】
- 控制面板采用双栏分区布局，显示与画面、构图辅助、线条外观、截屏与系统各自独立成区，正常窗口尺寸下无需滚动。
- 增加复位/重置功能。
- 增加登录时自动启动选项。
- 增加“联系和打赏作者”，并列显示支付宝与微信打赏图片。
- 联系邮箱：xingheyaoshi@163.com。

【发布说明】
- 当前没有 Apple Developer ID 证书，构建产物采用 ad-hoc 临时签名，尚未经过 Apple 公证。
- 提供首次运行说明与辅助脚本。
- 提供 GitHub Actions 自动构建流程，便于在真实 macOS 环境生成并校验 Universal 应用。
