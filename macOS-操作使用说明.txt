《观格 for macOS》操作使用说明
版本：1.1.0
适用系统：macOS 13 Ventura 及以上
支持设备：Apple Silicon 与 Intel Mac

============================================================
一、安装与第一次运行
============================================================

1. 解压 GuanGe-macOS-Universal-1.1.0.zip。
2. 建议把“观格.app”拖进“应用程序”文件夹。
3. 因当前版本没有 Apple 开发者证书，第一次运行请右键单击“观格.app”，选择“打开”。
4. 如果系统仍提示无法验证开发者：
   打开“系统设置”→“隐私与安全性”→向下找到观格→单击“仍要打开”。
5. 也可以在解压目录内右键“首次运行.command”并选择“打开”。此脚本只会移除观格自身的下载隔离标记，然后启动观格。

注意：请只从作者提供的仓库或可信来源取得程序。未使用 Developer ID 签名和 Apple 公证的应用，macOS 无法替您确认发布者身份。

============================================================
二、开始使用
============================================================

启动后会出现“观格 · 控制面板”，同时在菜单栏显示取景框形状的观格图标。
参考层会覆盖在画面上方，但它会忽略鼠标事件，所以仍可正常点击播放器、网页和其他窗口。
关闭控制面板不会退出观格。需要退出时，请单击菜单栏的观格图标，然后选择“退出观格”。

所有设置即时生效，不需要单击“应用”按钮。
控制面板顶部可通过“简体中文 | English”即时切换界面语言。

============================================================
三、显示与画面
============================================================

1. 显示范围
   - 全部显示器：每台显示器都显示参考线。
   - 主显示器：只在主屏显示。
   - 指定显示器：选择一台指定屏幕。

2. 画幅比例
   - 适配屏幕
   - 1:1、4:3、3:2、16:10、16:9、18:9、21:9
   - 1.85:1、2.35:1、2.39:1
   - 4:5、3:4、9:16

3. 轮巡画幅快捷键
   - 默认 Shift + ↑：切换到上一个画幅。
   - 默认 Shift + ↓：切换到下一个画幅。

============================================================
四、构图辅助
============================================================

主构图线包括：
三分法/九宫格、二等分、四等分网格、五等分网格、黄金分割网格、中心十字、黄金三角形（左/右）、黄金螺旋（四个方向）、动态对称。

可另外开启“对角线”，它会叠加在主构图线上。

可另外开启“安全框”：
1. 面板以 1%–99% 调整，默认 90%。
2. 背后的实际绘制采用四边相同的像素内距。
3. 例如上边向内 10 像素时，下边、左边和右边也都向内 10 像素，不会因画幅宽高不同而出现边距不一致。
4. 安全框颜色可以单独设置。

轮巡主构图线快捷键：
默认 Control + ↑ 为上一个，Control + ↓ 为下一个。

切换画幅比例或主构图线时，当前名称会在屏幕中央快速淡入，约 0.3 秒后快速淡出。提示使用 macOS 系统无衬线字体、透明背景且不播放音效。

============================================================
五、线条外观
============================================================

1. 主构图线和画幅边框均为实线，不使用描边色。
2. 可选择预设颜色，也可单击颜色块自定义颜色。
3. 可调整线条宽度与透明度。
4. 默认 Option + ↑ / Option + ↓ 可轮巡切换线条颜色。

============================================================
六、截图
============================================================

默认截图快捷键：Control + Option + S。
截图会包含当前屏幕画面和观格参考线；控制面板与打赏窗口会在截图时临时隐藏，完成后自动恢复。
截图成功后会播放清脆的系统快门提示声，并让目标屏幕轻微快闪一下；不会弹窗，也不会发送任何系统通知。
如果显示范围是“全部显示器”，观格会为每台显示器分别保存一张 PNG 图片。

第一次截图时，macOS 会要求“屏幕录制”权限。若截图失败：
1. 打开“系统设置”→“隐私与安全性”→“屏幕录制”。
2. 允许观格录制屏幕。
3. 退出并重新启动观格后再试。

可在控制面板输入或浏览选择截图目录，“打开目录”会直接在访达中打开它。

截图文件名规则支持以下变量：
{date} 日期（yyyyMMdd）
{time} 时间（HHmmss）
{frame} 当前画幅比例
{guide} 当前主构图线
{display} 显示器名称编号
{index} 显示器序号

默认规则：GuanGe-{date}-{time}
如果同时截取多台显示器但规则中没有 {display} 或 {index}，观格会自动补充显示器编号。文件名中的 macOS 非法字符会自动替换，重名文件会自动增加数字序号。

受版权保护的 DRM 视频可能禁止系统截屏，画面可能显示为黑色。这是播放平台或 macOS 的保护限制，观格无法绕过。

============================================================
七、自定义快捷键
============================================================

1. 单击当前快捷键框。
2. 框内显示“请按新快捷键…”后，直接按新的按键组合。
3. 新快捷键会立即替换旧快捷键并生效。
4. 按 Esc 可取消本次录入。
5. 如果快捷键被系统或其他软件占用，控制面板会显示提示，请换一个组合。

默认快捷键：
显示/隐藏参考线        Control + Option + G
截图                   Control + Option + S
显示/隐藏控制面板      Control + Option + P
上一个/下一个画幅      Shift + ↑ / Shift + ↓
上一个/下一个构图线    Control + ↑ / Control + ↓
上一个/下一个线条颜色  Option + ↑ / Option + ↓

============================================================
八、其他设置
============================================================

登录时自动启动：建议先把观格.app 放入“应用程序”文件夹，再开启该选项。
复位/重置：恢复画幅、构图线、颜色、截图目录和全部快捷键的初始值。
联系和打赏作者：显示支付宝与微信图片，以及作者联系邮箱。
“联系和打赏作者”入口位于控制面板正中下方，并与底部四个操作按钮居中对齐。
