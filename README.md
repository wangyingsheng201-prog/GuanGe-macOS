# 观格 for macOS

观格是一款用于分析电影、视频和图片构图的桌面参考线工具。它会在画面上方显示透明的构图参考线，但不会拦截鼠标点击，因此不影响播放器、浏览器和其他软件的正常操作。

本仓库是观格的原生 macOS 版本，采用 Swift、SwiftUI 与 AppKit 编写，不依赖 Electron 或第三方运行库。

## 系统要求

- macOS 13 Ventura 或更高版本
- 同时支持 Apple Silicon（M1/M2/M3/M4 等）与 Intel Mac
- 截图功能需要用户授予“屏幕录制”权限

## 主要功能

- 全屏透明、鼠标穿透的构图参考层，可跟随全屏视频空间
- 全部显示器、主显示器或指定显示器
- 1:1、4:3、3:2、16:10、16:9、18:9、21:9、1.85:1、2.35:1、2.39:1、4:5、3:4、9:16 等画幅
- 三分法、二等分、多级网格、黄金分割网格、中心十字、黄金三角形、四向黄金螺旋、动态对称等构图线
- 可独立开关的对角线与安全框
- 安全框在四边采用相同像素内距；面板仍以 1%–99% 显示，默认 90%
- 实线、无描边，可调颜色、宽度和透明度
- 截图包含当前画面和参考线；多显示器模式会为每台显示器各保存一张
- 截图成功后仅播放清脆快门声并轻微闪屏，不弹窗、不发送系统通知
- 切换画幅或主构图线时，在屏幕中央短暂显示当前名称
- 支持简体中文与 English 即时切换
- 支持使用日期、时间、画幅、构图线和显示器变量自定义截图文件名
- 所有设置即时生效；所有全局快捷键都可直接录入替换
- 快捷键轮巡画幅、主构图线和线条颜色，分别支持上一个与下一个
- 菜单栏常驻、开机启动、重置、打开截图目录、联系和打赏作者

## 默认快捷键

| 功能 | 默认快捷键 |
|---|---|
| 显示/隐藏参考线 | Control + Option + G |
| 截图 | Control + Option + S |
| 显示/隐藏控制面板 | Control + Option + P |
| 上一个/下一个画幅 | Shift + ↑ / Shift + ↓ |
| 上一个/下一个主构图线 | Control + ↑ / Control + ↓ |
| 上一个/下一个线条颜色 | Option + ↑ / Option + ↓ |

单击控制面板里的快捷键框，然后直接按下新组合即可替换。按 Esc 可取消录入。

## 构建

在安装了 Xcode 15 或更高版本的 Mac 上执行：

```bash
bash scripts/build-app.sh
```

构建结果：

```text
dist/GuanGe-macOS-Universal-1.1.0.zip
```

也可以在 GitHub 仓库的 Actions 页面手动运行“构建观格 macOS Universal 版本”，完成后从构建产物中取得 zip。

## 关于签名

当前项目没有 Apple Developer ID 证书，因此构建脚本只进行 ad-hoc 临时签名，不能完成 Apple 公证。第一次打开时，请先尝试右键“观格.app”并选择“打开”；若仍被阻止，请前往“系统设置 → 隐私与安全性”并选择“仍要打开”。发布给更多普通用户前，建议后续申请 Apple Developer Program，对应用进行 Developer ID 签名与公证。
